Installation Guide
Using The Table Inside BrideWorship
A
After installing The Table, open it alongside BrideWorship Pro using Window Snap (Windows Win+←/→) or Split View (Mac).
B
Search any sermon in The Table. Select and copy the quote you want (Ctrl+C / ⌘C).
C
Click "Add to Queue" tab here in BrideWorship → paste the text → choose slide-split mode → Project Now.
D
Or use the Built-in Sermon Reader tab — browse and project directly without switching apps.
💡 No install needed: Use the Browse Sermons tab above to access branham.org directly inside BrideWorship Pro, or the Built-in Reader tab to paste and project sermon quotes without The Table app.
